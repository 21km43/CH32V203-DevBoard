# CH32V203 Development Board

## Summary
Schematic and Board layout of CH32V203 Development Board. This project was created using [KiCAD](https://www.kicad.org/).

## MCU
CH32V203C8T6  
144MHz, 20KB RAM, 64KB ROM  
[Datasheet](https://akizukidenshi.com/goodsaffix/CH32V203.pdf)

## Features
* CH32V203C8T6 RISC-V
* `144 MHz` maximum frequency, `64 KB` of Flash memory, `20 KB` of SRAM
* `8 MHz` crystal (High-Speed Enternal Clock)
* `32.768 KHz` crystal (Low-Speed External Clock)
* Red LED for `PC13`
* Tactile switch for `PA0`
* 2x20 side pins
* Size: `50.80 mm x 20.32 mm`
* USB Device (Type-C)
  * PA11 USB1_DM
  * PA12 USB1_DP
* USB Host (Type-C)
  * PB6 USB2_DM
  * PB7 USB2_DP
